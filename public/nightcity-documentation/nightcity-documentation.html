3. HTML structure

All HTML code is divided into three logical unit: header, content, footer

All js scripts are connected at the bottom of the document, which provides fast loading and displaying the site.

 

 

4. How to change the text in the sections?

Any text on the page is easy to change using notepad.

  1. In Google Chrome press F12 (Cmd + Opt + I for Mac)
  2. Clickany element on the page and Developer Tools highlights the html code that needs to be corrected in index.html
  3. Open index.html in Notepad and change the text. That's all!

5. How to change images?

All pictures are located in the “img”. You can replace the desired pictures on their own and they are automatically changed on the site.

6. How to change icons?

For the best display on the Retina display, we use the Font Awersome http://fontawesome.io/icons/ just change the class “fa-*” to one from list.
